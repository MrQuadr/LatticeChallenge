## [Lattice Challenge](https://www.latticechallenge.org/)

### Описание

Опираясь на известную статью Айтаи [1], организаторы построили базисы решёток, для которых решение задачи SVP (поиска кратчайшего вектора) влечёт за собой решение SVP **во всех** решётках некоторой меньшей размерности. Это не означает, что можно одновременно решить все экземпляры, — речь идёт о том, что данные базисы позволяют решать даже задачи наихудшего случая. Авторы считают эти базисы решёток трудными экземплярами, наиболее подходящими для тестирования и сравнения современных алгоритмов редукции решёток.

Построение этих базисов решёток и доказательство существования коротких векторов в каждой из соответствующих решёток изложены в работе [2].


## [SVP Challenge](https://www.latticechallenge.org/svp-challenge/index.php)

### Описание

Вызов SVP помогает оценивать мощность алгоритмов поиска кратчайшего вектора и служит для сравнения различных типов алгоритмов — таких как просеивание (sieving) и перебор (enumeration). Представленные здесь решётки являются случайными решётками в смысле Голдстейна и Майер, и используются для тестирования алгоритмов решения задачи кратчайшего вектора (SVP) в евклидовых решётках.

## [Ideal Lattice Challenge](https://www.latticechallenge.org/ideallattice-challenge/index.php)

### Описание

Идеальные решётки представляют особый интерес, поскольку они лежат в основе кольцевых схем постквантовой криптографии (Ring-LWE, NTRU и др.).
Представленные здесь решётки дают возможность тестировать алгоритмы SVP и редукции решёток для **идеальных решёток**. В большинстве размерностей доступны несколько решёток, принадлежащих различным циклотомическим многочленам (различаемым по индексу циклотомического). Больше информации представлено в [3].

## [LWE Challenge](https://www.latticechallenge.org/lwe_challenge/challenge.php)

### Описание

Задачи обучения с ошибками (Learning With Errors, LWE).

Задача LWE состоит в восстановлении секретного вектора **s**, имея на входе экземпляр (**A**, **b**), где **A** — матрица размером *m×n* над ℤ_q, а **b** = **As** + **e** — вектор длины *m* над ℤ_q. Матрица **A** и целевой вектор **s** выбираются равномерно случайно, тогда как вектор ошибок **e** — из гауссовского распределения с параметром σ.

Представлены образцы экземпляров для тестирования алгоритмов решения задачи LWE. Главная цель вызова - помочь оценить практическую сложность задачи LWE, а также обеспечить сравнение различных типов LWE-решателей: метода Бабаи "ближайшая плоскость" [4], алгоритма BKW [5] и сведения к задаче кратчайшего вектора [6].

Поскольку, согласно современным результатам, сложность экземпляров LWE определяется главным образом «относительным размером ошибки» *α = σ/q* и размерностью *n*, страница предоставляет экземпляры LWE для широкого диапазона *α* и *n*.

Все вызовы созданы с использованием протокола многостороннего вычисления, выполненного совместно Эйндховенским технологическим университетом, Калифорнийским университетом в Сан-Диего и Техническим университетом Дармштадта. Протокол гарантирует, что ни одна из участвующих сторон не располагает дополнительной информацией об ответах.
